Step-by-Step Guideline

3 PHASES
1

Phase I: Preparation

Abrasive blast cleaning to ISO 8501-1 Sa 2.5 standard. Ensure surface is free of oil, grease, and mill scale. Degrease with alkaline cleaners if necessary.

  • check_circle Chloride testing
  • check_circle Profile depth verification
2

Phase II: Application

Apply Epoxy Zinc-Rich Primer (DFT 60μm). Follow with High-Build MIO Epoxy Intermediate coat (DFT 150μm) within 24-hour recoat window.

Tech Note

Monitor dew point; ensure substrate temp is at least 3°C above dew point.

3

Phase III: Finishing

Final coat of Polyurethane Finish (DFT 50μm). Provides UV resistance and color retention. Full chemical cure reached after 7 days at 20°C.
