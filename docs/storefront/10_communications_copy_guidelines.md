## 1. Brand Voice & Tone

### 1.1 Brand Personality

```
Pavlicevits Brand Personality:
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  🎯 EXPERT         "We know paints inside and out"                     │
│  🤝 HELPFUL        "We're here to guide you"                           │
│  💪 TRUSTWORTHY    "40+ years of reliability"                          │
│  🏠 LOCAL          "Your neighbor in Thessaloniki"                     │
│  ⚡ EFFICIENT      "Quality without complexity"                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Voice Characteristics

| Characteristic | What it means | Example |
|----------------|---------------|---------|
| **Knowledgeable** | We share expertise confidently | "Το 2K βερνίκι προσφέρει ανώτερη αντοχή και γυαλάδα" |
| **Approachable** | Technical but not intimidating | "Δεν ξέρετε τι χρειάζεστε; Ρωτήστε μας!" |
| **Direct** | Clear, no fluff | "Δωρεάν αποστολή για αγορές άνω των €50" |
| **Supportive** | Always ready to help | "Είμαστε εδώ για οποιαδήποτε απορία" |
| **Local** | Thessaloniki-proud | "Εξυπηρετούμε τη Θεσσαλονίκη από το 1985" |

### 1.3 Tone Spectrum

```
Formal ←───────────────●───────────────→ Casual
                       ↑
                  Pavlicevits
              (Professional but warm)
```

| Context | Tone Adjustment |
|---------|-----------------|
| Product descriptions | Professional, informative |
| Marketing emails | Warm, engaging |
| Error messages | Apologetic, helpful |
| Customer support | Empathetic, solution-focused |
| Legal/Terms | Formal, clear |
| Social media | Friendly, conversational |

### 1.4 Language Guidelines

**Greek (Primary):**
- Use formal "εσείς" (εσύ only in casual contexts)
- Avoid overly academic Greek
- Use common technical terms (not archaic translations)
- Maintain consistent terminology

**English:**
- British English spelling (colour, honour)
- Clear, simple sentences
- Avoid idioms that don't translate
- Keep consistent with Greek messaging

### 1.5 Words to Use vs. Avoid

**Use:**
| Greek | English | Why |
|-------|---------|-----|
| Αξιόπιστο | Reliable | Builds trust |
| Επαγγελματικό | Professional | Quality focus |
| Εύκολο | Easy | Reduces friction |
| Δωρεάν | Free | Universal appeal |
| Γρήγορο | Fast | Efficiency |
| Εγγύηση | Guarantee | Confidence |

**Avoid:**
| Avoid | Why | Use Instead |
|-------|-----|-------------|
| Φτηνό (Cheap) | Implies low quality | Οικονομικό (Affordable) |
| Αμφιβολία (Doubt) | Negative | Ερώτηση (Question) |
| Πρόβλημα (Problem) | Negative | Ζήτημα (Issue), Απορία (Query) |
| Υποχρεωτικό (Mandatory) | Harsh | Απαραίτητο (Required) |
| Απαγορεύεται (Forbidden) | Harsh | Δεν επιτρέπεται (Not allowed) |

## 4. Product Description Guidelines

### 4.1 Product Description Structure

```
STRUCTURE:
1. Opening hook (1 sentence) - Immediate benefit
2. Key features (3-5 bullet points)
3. Technical specifications (table or list)
4. Use cases / Ideal for
5. Pro tip or recommendation (optional)
```

### 4.2 Product Description Template

**Greek Template:**
```
[Όνομα Προϊόντος]

[Εναρκτήρια πρόταση που αναδεικνύει το κύριο πλεονέκτημα]

Κύρια Χαρακτηριστικά:
• [Χαρακτηριστικό 1 + όφελος]
• [Χαρακτηριστικό 2 + όφελος]
• [Χαρακτηριστικό 3 + όφελος]

Ιδανικό για:
[Περιγραφή χρήσεων]
```

```
[Product Name]

• [Feature 1 + benefit]
• [Feature 2 + benefit]
```

### 4.3 Sample Product Descriptions

**HB Body P961 Clear Coat 2K HS:**

**Greek:**
```
HB Body P961 Clear Coat 2K HS

Επαγγελματικό βερνίκι υψηλών στερεών που προσφέρει εξαιρετική 
γυαλάδα και αντοχή στις γρατσουνιές.

Κύρια Χαρακτηριστικά:
• Υψηλή περιεκτικότητα σε στερεά (~52%) για βαθιά γυαλάδα
• Ταχεία στέγνωμα σε 20-30 λεπτά
• Εξαιρετική αντοχή σε UV και χημικά
• Εύκολη εφαρμογή με αναλογία μίξης 2:1
• Συμβατό με όλα τα basecoats HB Body

Ιδανικό για:
Επαγγελματικές εφαρμογές σε αυτοκίνητα, μοτοσικλέτες και οχήματα. 
Κατάλληλο τόσο για ολική βαφή όσο και για τοπικές επισκευές.

💡 Pro Tip: Για μέγιστη γυαλάδα, εφαρμόστε 2-3 λεπτές στρώσεις 
με 10 λεπτά μεταξύ τους. Χρησιμοποιήστε τον σκληρυντή P962 για 
βέλτιστα αποτελέσματα.
```

**English:**
```
HB Body P961 Clear Coat 2K HS

Professional high solid clear coat that delivers exceptional gloss 
and scratch resistance.

Key Features:
• High solid content (~52%) for deep gloss
• Fast drying in 20-30 minutes
• Excellent UV and chemical resistance
• Easy application with 2:1 mixing ratio
• Compatible with all HB Body basecoats

Ideal for:
Professional applications on cars, motorcycles, and vehicles. 
Suitable for both full resprays and spot repairs.

💡 Pro Tip: For maximum gloss, apply 2-3 thin coats with 10 minutes 
between each. Use P962 hardener for optimal results.
```
