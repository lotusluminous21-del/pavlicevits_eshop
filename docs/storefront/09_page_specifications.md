### 1.3 Content Requirements

| Section | Content | Source |
|---------|---------|--------|
| Hero | Headline, subheadline, 2 CTAs, background image | CMS/Static |
| Trust Bar | 4 icons with text | Static |
| Categories | 6-8 category cards with images | Shopify Collections |
| Color Finder CTA | Banner with CTA | Static |
| Featured Products | 5-10 products with tabs | Shopify Products (filtered) |
| Brands | 6-10 brand logos | Static/Metafields |
| Newsletter | Form, incentive text | Static + API |
| Footer | Links, contact, social, payment icons | Static |
