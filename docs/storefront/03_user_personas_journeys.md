## 1. Persona Development Overview

### 1.1 Research Basis

These personas are developed based on:
- Greek automotive and paint market demographics
- E-commerce behavior patterns in Greece (2025-2026)
- HBBody.com and competitor customer analysis
- Thessaloniki regional market characteristics

### 1.2 Persona Summary Matrix

| Persona | Segment | Age | Visit Frequency | Avg Order Value | Priority |
|---------|---------|-----|-----------------|-----------------|----------|
| **Δημήτρης** | DIY Homeowner | 35-50 | 2-3x/year | €25-50 | High |
| **Νίκος** | Professional Painter | 28-45 | Weekly | €80-150 | Critical |
| **Μαρία** | Shop Owner | 40-55 | 2-3x/week | €300-800 | Critical |
| **Γιώργος** | Car Enthusiast | 22-35 | Monthly | €50-120 | Medium |
| **Αλέξανδρος** | Boat Owner | 45-65 | Seasonal | €150-400 | Medium |

### 11.3 Product Comparison Feature

```typescript
interface ProductComparisonConfig {
  maxProducts: 4;
  displayFields: [
    'image',
    'name',
    'brand',
    'price',
    'rating',
    'type', // 1K/2K
    'solidContent', // %
    'dryingTime',
    'potLife',
    'coverage', // m²/L
    'voc', // g/L
    'glossLevel',
    'compatiblePrimers',
    'mixingRatio',
  ];
  highlightBest: true; // Highlight best value per field
  showDifferencesOnly: false; // Toggle option
}
```
